# 超怪诞翻牌 (Super Weird Flip Cards)

一款发布于 TikTok 的轻量级翻牌记忆小游戏 —— 纯 HTML/CSS/JS 单文件,零依赖,浏览器双击即玩。

## 玩法
- 开局先亮牌若干秒供记忆,翻回后开始限时(每关 60s)。
- 每次翻两张,相同配对保留,不同翻回。
- 失败条件:可错次数用尽 或 时间到。
- 关卡:4×4 ×4 关(失误 3→2→1→0)→ 6×6 ×2 关,逐关变难。
- 每关 1 次「提示」(亮出任意一组对子)。

## 运行
直接用浏览器打开 `index.html`;或起本地服务器:
```bash
python3 -m http.server 8000
# 浏览器访问 http://127.0.0.1:8000/index.html
```

## 目录
- `index.html` — 游戏(单文件:逻辑 + UI + 样式)
- `back_img/` — 背景图(首页 / 闯关页)
- `Cards/` — 卡面素材
- `sound/` — 背景音乐与音效
- `翻牌记忆_PRD.md` — 产品需求文档

## 技术
纯 H5(无引擎/无构建)。卡面、背景、音效均可替换:卡面改 `FACE_POOL`,背景改 CSS 中的 `background`,关卡改 `LEVEL_PLAN`。
